```c
函數 EvaluateInfix(表達式):
    創建 數字堆疊 (values)
    創建 運算子堆疊 (ops)

    對於 表達式中的每個標記 (token):
        如果 token 是 數字:
            將其壓入 values 堆疊

        否則如果 token 是 '(':
            將其壓入 ops 堆疊

        否則如果 token 是 ')':
            當 ops 頂部不是 '(' 時:
                執行 ApplyOperator(values, ops)
            彈出 ops 頂部的 '('

        否則如果 token 是 運算子 (+, -, *, /):
            當 ops 不為空 且 ops 頂部優先權 >= 當前 token 優先權:
                執行 ApplyOperator(values, ops)
            將當前 token 壓入 ops 堆疊

    當 ops 不為空時:
        執行 ApplyOperator(values, ops)

    返回 values 堆疊的最後一個數字 (即結果)

函數 ApplyOperator(values, ops):
    運算子 = 從 ops 彈出
    右操作數 = 從 values 彈出
    左操作數 = 從 values 彈出
    結果 = 根據 運算子 計算 (左操作數, 右操作數)
    將 結果 壓入 values 堆疊
```
